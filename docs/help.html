Help & Frequently Asked Questions (FAQ)

How do I find a cheatsheet?

You can use the **search bar** in the navigation to filter by title, or browse by **category** using the dropdown menu.

How do I contribute a new cheatsheet?

Please refer to the Contributing section in the README for detailed instructions, including using our standardized Markdown template.

What if I find an error in a cheatsheet?

We encourage you to open an issue on GitHub to report the error, or even better, submit a pull request with the correction!

Can I request a new cheatsheet topic?

Absolutely! You can open a new issue on our GitHub repository with the label "feature request" or "cheatsheet request".

Is this project open source?

Yes! This project is open source and licensed under the MIT License. Contributions are highly welcomed.
